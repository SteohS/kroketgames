# Animal images

One PNG per animal, transparent background preferred, square-ish, ≥ 512px.

Naming: `<animal-id>.png` — ids must match `js/data/animals.js`:
cow.png, duck.png, dog.png, cat.png, sheep.png, chicken.png, horse.png, pig.png

Missing files are fine: the game falls back to an emoji automatically.

Suggested CC0 sources: kenney.nl (animal packs), openmoji.org, opengameart.org.
Keep files < 200 KB each (export at 512–1024px).
