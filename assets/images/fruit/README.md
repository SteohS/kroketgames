# Fruit images

One PNG per fruit, transparent background preferred, square-ish, ≥ 512px.

Naming: `<fruit-id>.png` — ids must match `js/data/fruit.js`:
apple.png, banana.png, orange.png, strawberry.png, kiwi.png, lemon.png,
cherry.png, peach.png, watermelon.png, mango.png, melon.png, grape.png,
pear.png, ananas.png, coconut.png, blueberries.png

Missing files are fine: the game falls back to an emoji automatically.
Fruit is a **soundless** category — no `assets/audio/fruit/` is needed.

Suggested CC0 sources: kenney.nl, openmoji.org, opengameart.org.
Keep files < 200 KB each (export at 512–1024px).
