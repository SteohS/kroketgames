# Object images

Everyday countable objects (for the counting game and future games).
One PNG per object, transparent background preferred, square-ish, ≥ 512px.

Naming: `<object-id>.png` — ids must match `js/data/countables.js`:
apple.png, banana.png, orange.png, strawberry.png, star.png, flower.png,
fish.png, leaf.png

Missing files are fine: the game falls back to an emoji automatically.

Keep files < 200 KB each (export at 512–1024px). See `docs/asset-prompts.md`
for the "Object / nature" prompt template so the style stays consistent.
